🎵 Vibe Radio
A sleek, minimalist web application that creates a curated sensory environment based on your mood. Built in a single 6-hour sprint using Vibe Coding principles.

🚀 The Concept
"Vibe Radio" allows users to select a theme (e.g., Cyberpunk, Rainy, Zen) and instantly transforms the entire interface. It synchronizes a live YouTube music stream with a dynamic CSS-animated background to create a fully immersive focus or relaxation station.

✨ Key Features
Dynamic Theme Engine: The entire site's color palette, gradients, and button accents shift instantly when a mood is selected.

Synced Music Streams: Integrated YouTube API to fetch high-quality, embed-friendly lo-fi and ambient livestreams.

Glassmorphism UI: A modern, frosted-glass design aesthetic that stays clean and distraction-free.

Responsive Design: Fully optimized for both desktop focus sessions and mobile "vibe" checks.

Vibe Coding Build: Developed using natural language prompting to bridge the gap between idea and execution in record time.
